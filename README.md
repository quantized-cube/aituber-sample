# AITuber

[AITuberを作ってみたら生成AIプログラミングがよくわかった件](https://bookplus.nikkei.com/atcl/catalog/23/10/31/01079/)

openai==1.30.5 対応版

## 仮想環境

```
# 作る（初回のみ）
python -m venv .venv
# 入る
.\.venv\Scripts\activate
# ライブラリのインストール（初回のみ）
pip install -r requirements.txt
# 抜ける
deactivate
```

## run.pyを実行する

```
python run.py
```

他の設定は上記書籍参照
